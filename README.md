# test_frask_react
[heroku](https://test-flask-and-react.herokuapp.com/)

## Getting started
```bash
$ npm install axios
$ yarn start
```

## Local
```bash
$ pip install -r requirements.txt
$ python app.py
```

## docker-compose
``` bash
$ docker-compose build
$ docker-compose up
$ docker-compose down
```

### front
``` bash
$ dce front sh
# 以下 docker 内ターミナル
$ cd front
$ yarn start
```
